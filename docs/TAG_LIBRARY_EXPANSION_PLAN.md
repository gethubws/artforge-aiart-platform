# 标签库与预览图库扩展计划

## 当前基线

- 80 个公开标签，11 个分类。
- 工作台按分类分页加载标签，每页 8 个；支持分类内中英文名称、说明和提示词搜索。
- 作品筛选、风格包编辑等非创作场景使用轻量标签选项，不加载完整提示词和图库。
- `tag.preview_image_url` 继续作为兼容封面字段。
- `tag_preview` 保存多图预览，标签详情按需加载图库。
- 当前共有 145 条预览记录；11 个原有风格标签均包含封面、人物、风景、物品 4 张图片。
- 光线、构图、色彩标签均已补充两张统一场景对比图。

## 多图预览规范

每个重点标签建议配置 3 至 5 张图片：

1. `COVER / GENERAL`：列表封面与综合效果。
2. `COMPARISON / CHARACTER`：固定人物对比场景。
3. `COMPARISON / LANDSCAPE`：固定风景对比场景。
4. `COMPARISON / OBJECT`：固定物品或产品对比场景。
5. 可选 `EXAMPLE / NIGHT` 或 `EXAMPLE / CLOSE_UP`：补充夜景、近景或复杂光照。

同一类标签应尽量使用相同主体、构图和基础提示词，只改变被比较的标签。这样用户看到的是标签差异，而不是随机画面差异。

## 扩图顺序

### 第一批：风格标签（已完成）

- 范围：现有 11 个风格标签。
- 目标：每个标签补人物、风景、物品 3 张对比图。
- 已新增：33 张。
- 原因：风格差异最依赖视觉示例，也是用户选择提示词时最有价值的一组。

### 第二批：光线、构图、色彩（已完成）

- 光线 7 个标签，每个补人物和室内/风景 2 张。
- 构图 6 个标签，每个补同一人物或建筑场景 2 张。
- 色彩 3 个标签，每个补同一静物和风景 2 张。
- 已新增：32 张。

### 第三批：主体、质量、氛围

- 主体标签重点补不同题材覆盖，不必强制做完全相同的对比场景。
- 质量标签重点展示局部细节和线条差异。
- 氛围标签使用同一环境做情绪对比。
- 预计新增量：20 至 30 张。

当前图库共有 145 张图片。完成第三批后预计达到 165 至 175 张，仍可由当前分页、懒加载和浏览器缓存结构稳定承载。

## 已完成标签扩展

当前已达到 80 个高质量标签，已完成的扩展包括：

1. 风格：版画、铅笔素描、漫画线稿、赛璐璐、黏土动画、低多边形、纸艺、复古海报。
2. 镜头：长焦、广角、鱼眼、微距、浅景深、移轴、动态模糊。
3. 材质：玻璃、金属、陶瓷、织物、木材、半透明、磨砂。
4. 时代与文化：蒸汽朋克、装饰艺术、新艺术、包豪斯、宋画、浮世绘、Y2K。
5. 环境：雨天、雪景、雾天、水下、太空、遗迹、沙漠、茂密森林。

后续不再以数量为主要目标，优先根据真实搜索无结果词、标签使用次数和常见组合数据决定新增内容。

新增标签前应满足：

- 与已有标签不是简单同义词。
- 能稳定改变生成结果。
- 有清晰中文名称、中文说明、正向词和必要的反向词。
- 至少有 1 张封面图，并列入后续多图补全队列。
- 能与主体、光线、构图等其他分类正常组合。

## 后续后台能力

- 管理员维护预览图顺序、场景类型、封面和提示词快照。
- 批量导入标签及图库元数据。
- 统计标签使用次数、搜索无结果关键词和标签组合频率。
- 根据真实使用数据决定下一批标签，而不是只按数量扩张。
