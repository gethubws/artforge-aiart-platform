# Changelog

本项目从 `v1.0.0` 起按语义化版本记录稳定节点。

## [1.3.1] - 2026-07-21

### Added

- 增加登录页、创作工作台、作品广场、风格包市场和任务市场截图。
- 增加 MIT 开源许可证。

### Changed

- 整理 README，使其适合作为公开仓库首页。
- 从公开仓库及完整 Git 历史中移除论文和早期代码实现文档。

## [1.3.0] - 2026-07-21

### Added

- 作品、风格包与任务市场的正式分页和页码导航。
- 大图、标准、紧凑三档内容密度，并持久化用户偏好。
- 风格包多图拼贴预览、详情主图切换和版本展示。
- 管理员标签预览图库维护、用户管理、运营图表与标签分析。
- 标签多场景预览、双语说明、收藏、订阅与通知链路。

### Changed

- 风格包明确为同一视觉风格下的多人共创作品集合。
- 重构作品广场搜索请求，加入防抖和竞态保护，加载时保留现有内容。
- 统一作品库、作品广场、风格包和任务页面的卡片布局与分页交互。

### Fixed

- 修复快速搜索时图片反复闪烁的问题。
- 修复切换分页数量后内容消失及卡片尺寸语义不清的问题。
- 修复风格包筛选参数丢失、多图内容缺失和详情排版异常。

## [1.2.1] - 2026-07-20

- 增加可重复执行的演示市场数据初始化脚本。

## [1.2.0] - 2026-07-20

- 重做管理员控制台并加入完整用户管理。

## [1.1.0] - 2026-07-20

- 增加标签图库管理、预览图操作与使用分析。

## [1.0.0] - 2026-07-20

- 固化可运行的平台基础版本。
