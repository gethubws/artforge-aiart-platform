# Changelog

本项目从 `v1.0.0` 起按语义化版本记录稳定节点。

## [1.4.0] - 2026-07-21

### Added

- 增加包内资源、不可变资源修订、资源分类与批量导入接口。
- 增加版本资源锁定表，历史版本不再受后续资源替换影响。
- 增加购买后资源清单、文件地址、提示词和生成参数访问控制。
- 增加资源包授权类型、授权摘要、商业使用标记和资源统计。
- 增加“翡翠童话森林”大型示例资源包生成与初始化脚本。

### Changed

- 风格包升级为类似游戏引擎资源商店的“风格资源包”，效果作品作为独立展示层保留。
- 风格资源市场成为登录后的默认首页。
- 市场卡片优先展示资源拼贴、资源数、类目数和版本信息。
- 协作者可维护包内资源，并能在“我的资源包”中找到参与协作的资源包。

### Fixed

- 将资源表迁移改为兼容 MySQL 8.0.44 的可重复执行脚本。

## [1.3.1] - 2026-07-21

### Added

- 增加登录页、创作工作台、作品广场、风格包市场和任务市场截图。
- 增加 MIT 开源许可证。

### Changed

- 整理 README，使其适合作为公开仓库首页。
- 从公开仓库及完整 Git 历史中移除论文和早期代码实现文档。

## [1.3.0] - 2026-07-21

### Added

- 作品、风格包与任务市场的正式分页和页码导航。
- 大图、标准、紧凑三档内容密度，并持久化用户偏好。
- 风格包多图拼贴预览、详情主图切换和版本展示。
- 管理员标签预览图库维护、用户管理、运营图表与标签分析。
- 标签多场景预览、双语说明、收藏、订阅与通知链路。

### Changed

- 风格包明确为同一视觉风格下的多人共创作品集合。
- 重构作品广场搜索请求，加入防抖和竞态保护，加载时保留现有内容。
- 统一作品库、作品广场、风格包和任务页面的卡片布局与分页交互。

### Fixed

- 修复快速搜索时图片反复闪烁的问题。
- 修复切换分页数量后内容消失及卡片尺寸语义不清的问题。
- 修复风格包筛选参数丢失、多图内容缺失和详情排版异常。

## [1.2.1] - 2026-07-20

## [1.0.0] - 2026-07-20
